## 🚀 使用方法

### 立即开始
```bash
# 一键启动完整系统
pnpm run quick-start

# 或者手动启动
pnpm run dev:start
pnpm run server:status
```

### 日常监控
```bash
# 检查服务器状态
pnpm run server:status

# 实时监控日志
pnpm run logs:monitor

# 分析日志文件
pnpm run logs:analyze
```

### 问题排查
```bash
# 查看服务器状态
pnpm run dev:status

pnpm run logs:view
```
